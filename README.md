# REST proxy service with dashboard

A monorepo containing a REST proxy server (Express.js) and a Frontend (Next.js) designed for auditing api calls.

### Tech Stack

- Node js
- Express
- Next js
- Socket io

---

### Running locally

_Install packages_

```bash
npm install
```

_Run both apps_

```bash
npm run start
```

Default frontend url: [localhost:3000](http://localhost:3000)
